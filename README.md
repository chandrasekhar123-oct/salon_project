# Salon Booking Web App

A modern salon booking application built with Flask and SQLAlchemy.

## Features
- **User Authentication**: Secure signup and login.
- **Service Browsing**: Explore salons and services.
- **Booking System**: Select services and book slots.
- **Responsive Design**: Premium UI optimized for mobile and desktop.

## Local Setup (Windows)

1. **Run Setup**:
   Double-click `setup.bat`. This will:
   - Create a virtual environment (`venv`).
   - Install all required Python packages.

2. **Launch App**:
   Double-click `run.bat`. This will:
   - Start the Flask development server at `http://127.0.0.1:5000`.
   - Automatically initialize the database (`salon.db`).

## Technologies Used
- **Backend**: Python, Flask
- **Database**: SQLite, SQLAlchemy
- **Frontend**: HTML5, Vanilla CSS3, JavaScript
- **Auth**: Flask-Login, Werkzeug Security
